# Social Dashboard

Modern TYPO3 extension for analyzing Meta, Facebook and Instagram insights in one dashboard.

## Overview

Social Dashboard is a TYPO3 extension that connects with the Meta Graph API to display social media insights inside an intuitive backend dashboard.

It is designed to help businesses and content creators monitor performance, engagement and analytics directly from TYPO3.

## Features

- 📈 Dashboard with key performance metrics
- 📊 Engagement score calculation
- 📱 Facebook & Instagram integration
- 🔐 Meta OAuth authentication
- ⚡ AJAX data loading
- 🎯 Actionable optimization tips
- 💜 TYPO3 Backend Module

## Installation

Clone repository

composer install

Activate extension

Configure Meta API credentials

Flush TYPO3 cache

Enjoy 🚀

## Configuration

Copy

config/system/additional.example.php

to

config/system/additional.php

and insert your own Meta credentials.
